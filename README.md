<p align="center">Página inicial y formulário de registro de vídeos de AluraPlay, una plataforma de intercambio de vídeos.</p>


## Para correr esta aplicación siga los siguientes pasos 

* Ejecute el comando:  json-server --watch db.json --port 3001
* Corrar Go Live
* Sino tiene instalado json-server primero debe instalarlo en su PC


## Tecnologias utilizadas durante el curso
* Javascript
* NodeJS
* Json-server

## Tecnologias utilizadas en el projeto

* CSS
